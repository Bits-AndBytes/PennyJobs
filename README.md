# PennyJobs
A shared repository by Bits and Bytes for the PennyJobs Web Application

## Technologies needed
  - [Java SE 15](https://www.oracle.com/java/technologies/javase/jdk15-archive-downloads.html)
  - [Eclipse IDE for Enterprise Java and Web Developers](https://www.eclipse.org/downloads/packages/release/2021-09/r/eclipse-ide-enterprise-java-and-web-developers)
  - [SpringBoot / Spring Tools](https://spring.io/tools)
  - [Project Lombok](https://projectlombok.org/download)
  - [Thymeleaf](https://www.thymeleaf.org/download.html)
